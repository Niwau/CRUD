# CRUD API 🍖

Primeiro crud feito por mim usando MongoDB e NodeJs. Basicamente simula uma TodoList.

## Funcionalidades 🍔

- Criar tarefas.
- Ler tarefas.
- Atualizar tarefas.
- Deletar tarefas.

## Stack utilizada 🧀

**Back-end:** Node, Express, MongoDB
